3

확장 설치하기 — 두 가지 방법

코드까지 자동 업로드하려면 크롬 확장이 필요해요. 가장 쉬운 웹스토어 또는 GitHub 릴리스에서 개발자 모드로 설치할 수 있습니다.

가장 쉬움

A · 크롬 웹스토어

설치 후 자동 업데이트까지 알아서.

  1. 크롬 웹스토어에서 “7일 7솔” 검색
  2. Chrome에 추가 → 확인
  3. 새 버전은 크롬이 자동으로 갱신
참고 새 버전이 심사 중이면 스토어 반영이 늦을 수 있어요. 최신 기능을 바로 쓰려면 오른쪽 개발자 모드로 설치하세요.
최신 · 심사 대기 없음

플랫폼 연동 + 확장 토큰

스터디 안에서 확인하기

대시보드에서 누가 · 무엇을 · 어떻게(코드) 풀었는지, 이번 기간 목표를 채웠는지, 벌금은 얼마인지 한눈에 봅니다.

  • 멤버별 이번 기간 진행도와 목표 달성 여부
